# Responsive Real Estate Website

## Project Overview
This is a responsive real estate website built using HTML, CSS, and basic JavaScript.
The website includes a header, hero section, featured properties, about us section,
contact form, and footer.

## Features
- Responsive design (Mobile, Tablet, Desktop)
- Featured property cards with images
- Contact form with basic JavaScript validation
- Clean and simple UI

## Technologies Used
- HTML5
- CSS3 (Flexbox)
- JavaScript (Basic)

## Challenges Faced
- Making the layout responsive
- Aligning property cards properly
- Adding basic form validation using JavaScript
